## 🧪 交互式刷题

双击 `quiz.html` 在浏览器中打开，或访问 **https://microzhai.github.io/quiz/**（需开启 GitHub Pages）。

### 功能

- 🎲 随机刷题 / 按主题筛选 / 按题型练习
- 📋 模拟考试（可选 20/50/100/自定义题数，按比例出卷）
- 📝 错题本（自动收录，localStorage 持久化）
- 📱 手机/平板/电脑 均可使用
- 🔌 离线可用（file:// 协议直接打开）

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `复习题整合.md` | 📖 唯一复习文档，含全部题目和答案 |
| `quiz.html` | 🧪 交互式刷题网页 |
| `index.html` | 同上（GitHub Pages 入口） |
| `questions.js` | 题库数据（quiz.html 加载） |
| `questions.json` | 题库数据（JSON 备份） |
| `parse_all.py` | 从 md 重新生成题库的脚本 |

## 🔧 维护

修改 `复习题整合.md` 后，运行以下命令重新生成题库：

```bash
python parse_all.py
```
