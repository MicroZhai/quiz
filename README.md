# 🔥 热处理考试刷题系统

交互式网页刷题工具，用于热处理考试复习。包含 678 道题目，涵盖 6 种题型、5 个知识模块。

## 🚀 使用方法

**直接双击 `quiz.html` 即可在浏览器中打开使用**，无需服务器。

## 📊 题库概况

| 题型 | 数量 | 判分方式 |
|------|:----:|------|
| 选择题 | 268 | 自动判分 |
| 判断题 | 254 | 自动判分 |
| 填空题 | 120 | 模糊匹配自动判分 |
| 简答题 | 15 | 显示答案自评 |
| 计算题 | 20 | 显示答案自评 |
| 作图题 | 1 | 显示答案自评 |

## 📂 知识模块

- 热处理基础知识
- 热处理设备与工艺
- 质量管理
- 仪表知识
- 综合复习

## ✨ 功能

- 🎲 **随机刷题** — 打乱全部题目自由练习
- 📂 **按主题/题型筛选** — 针对性练习薄弱环节
- 📋 **模拟考试** — 按比例出卷（20/50/100题或自定义），一页卷面答题，交卷打分
- 📝 **错题本** — 自动收录错题，集中重做，localStorage 持久化
- 📊 **正确率追踪** — 实时统计答题进度
- 📱 **响应式设计** — 手机/平板/电脑都能用
- 🔌 **离线可用** — 纯静态页面，`file://` 协议直接打开

## 📁 文件说明

| 文件 | 用途 |
|------|------|
| `quiz.html` | 单文件刷题网页 |
| `questions.js` | 题库数据（网页直接加载） |
| `questions.json` | 题库数据（JSON 格式备份） |
| `parse_all.py` | 解析脚本（从 .md 重新生成题库） |
| `*.md` | 原始复习资料 Markdown 文件 |

## 🔧 重新生成题库

```bash
python parse_all.py
```

会读取所有 `.md` 文件，重新解析并生成 `questions.json` 和 `questions.js`。
